## 二、快速开始

### 2.1 交互式菜单

```bash
cd "/mnt/e/Claude Code/projects/sysops-toolkit"
./sysops.sh
```

进入菜单后显示 9 个选项，按编号选择功能。

### 2.2 一键巡检

```bash
./sysops.sh --auto
```

自动运行全部 4 个模块并输出结果，包含系统健康评分。

### 2.3 巡检 + HTML 报告

```bash
./sysops.sh --auto --html
```

报告生成在 `reports/` 目录下，用浏览器打开。

### 2.4 运行单个模块

```bash
./sysops.sh --module 1    # 系统性能监控
./sysops.sh --module 2    # 用户活动追踪
./sysops.sh --module 3    # 文件系统扫描
./sysops.sh --module 4    # 日志分析
```

### 2.5 查看帮助

```bash
./sysops.sh --help
```

## 三、模块功能说明

### 模块一：系统性能监控仪

**功能**：
- CPU 整体使用率 + 各核心使用率
- 1/5/15 分钟系统负载
- 物理内存 + Swap 使用率（超阈值红色告警）
- CPU / 内存 Top 5 进程排行
- ASCII 负载趋势图（10 秒采样）

**技术原理**：`/proc/stat` 两次采样差值法

### 模块二：用户活动追踪器

**功能**：
- 当前登录会话（用户/终端/登录时间/来源IP）
- 历史登录记录（`last` 命令 + 用户登录次数统计）
- 登录失败统计（暴力破解 IP 排名）
- 暴力破解检测（300s 窗口/5 次阈值）
- sudo 操作审计
- sudoers 特权用户检查

**技术原理**：系统日志正则解析 + wtmp 二进制日志读取

### 模块三：文件系统扫描仪

**功能**：
- 磁盘空间检查（所有挂载点使用率，超阈值告警）
- 目录空间分析（du -sh 递归统计 Top 15）
- 大文件扫描（>100M，排除 /proc /sys /dev）
- 旧文件清理（>30 天，交互式确认删除）
- 权限安全审计（World-Writable / SUID / SGID）

**技术原理**：find 权限位匹配 + 八进制文件权限模型

### 模块四：日志分析引擎

**功能**：
- 实时日志追踪（tail -f + grep 高亮过滤）
- 日志分类统计（ERROR/WARN/INFO/DEBUG 四级分类）
- 主机名/服务名 Top 排名提取
- 按小时时间分布图
- 日志归档压缩（>7 天自动打包 .tar.gz）

**技术原理**：管道流式处理 + awk 结构化提取 + trap 信号捕获

### 模块五：主控调度中心

**功能**：
- dialog/whiptail 图形菜单
- 一键巡检模式
- HTML 报告生成（系统健康评分）
- 守护进程模式（后台定时巡检）
- crontab 定时任务配置

## 四、守护进程模式

### 启动

```bash
./sysops.sh --daemon 300    # 每 300 秒巡检一次
```

### 查看日志

```bash
tail -f logs/daemon_$(date +%Y%m%d).log
```

### 停止

```bash
./sysops.sh --daemon-stop
```

---

## 五、定时任务

### 配置

```bash
./sysops.sh --cron-setup 5   # 每 5 分钟自动巡检
```

### 移除

```bash
./sysops.sh --cron-remove
```

## 七、配置修改

编辑 `etc/config.conf` 调整阈值和参数：

```bash
CPU_THRESHOLD=80        # CPU 告警阈值
MEM_THRESHOLD=80        # 内存告警阈值
DISK_THRESHOLD=90       # 磁盘告警阈值
LARGE_FILE_SIZE="+100M" # 大文件阈值
OLD_FILE_MTIME="+30"    # 旧文件天数
BRUTE_FORCE_WINDOW=300  # 暴力破解检测窗口(秒)
BRUTE_FORCE_THRESHOLD=5 # 暴力破解触发次数
DAEMON_INTERVAL=300     # 守护进程巡检间隔(秒)
```
