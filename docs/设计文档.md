## 一、项目概述

### 1.1 项目名称

**Linux 系统运维工具箱的设计与实现**

### 1.2 项目目标

基于 Linux Shell 脚本编程，开发一个功能完整的系统监控与自动化运维工具包。通过实践验证操作系统核心概念（进程管理、文件系统、用户权限、I/O重定向、管道通信），建立"系统调用→内核机制→应用工具"的完整知识体系。

### 1.3 开发环境

## 三、模块详细设计

### 3.1 模块一：系统性能监控仪

**技术核心：`/proc` 文件系统解析**

```
/proc/stat    →  CPU 使用率 (整体 + 各核心，两次采样差值法)
/proc/loadavg →  1/5/15 分钟系统负载
/proc/meminfo →  物理内存 + Swap 使用率
ps aux        →  Top 5 进程排名
```

**关键算法：CPU 使用率两次采样差值**

1. 读取 `/proc/stat` 第一行 (cpu total)，记录 total_jiffies 和 idle_jiffies
2. sleep 0.5 秒
3. 再次读取，计算 delta_total 和 delta_idle
4. CPU% = (delta_total - delta_idle) / delta_total × 100

**各核心同理**：读取 `cpu0`, `cpu1`, ... 行，同样的两次采样差值法。

**阈值告警**：CPU > 80% 触发红色告警，内存 > 80% 触发红色告警。

### 3.2 模块二：用户活动追踪器

**技术核心：系统日志解析 + 正则匹配**

```
who -u              →  当前登录会话 (TTY, 登录时间, 来源IP)
last -n 20          →  历史登录记录 (/var/log/wtmp)
/var/log/auth.log   →  登录失败统计 + 暴力破解检测 + sudo 审计
/etc/sudoers        →  特权用户配置检查
```

**暴力破解算法**：
1. 从 auth.log 提取 "Failed password" 行，解析时间戳和来源IP
2. 筛选最近 300 秒内的记录
3. 同一 IP 失败次数 ≥ 5 → 标记为暴力破解，建议 `ufw deny`

### 3.3 模块三：文件系统扫描仪

**技术核心：find 命令高级用法 + 权限位运算**

```
df -h              →  磁盘配额与使用率
du -h --max-depth  →  递归目录大小统计
find -size +100M   →  大文件扫描
find -mtime +30    →  旧文件扫描 + 交互式删除
find -perm -o+w    →  全局可写文件检查
find -perm -4000   →  SUID 文件检查
find -perm -2000   →  SGID 文件检查
```

**权限位八进制判断**：
- `-perm -o+w`：other 有写权限 → 安全风险
- `-perm -4000`：SUID 位 → 执行时以文件所有者身份运行
- `-perm -2000`：SGID 位 → 执行时以文件所属组身份运行

### 3.4 模块四：日志分析引擎

**技术核心：管道通信 + 信号捕获 + 流式处理**

```
tail -f | grep --color   →  实时日志追踪 + 关键字高亮
awk + sort + uniq -c     →  按级别(ERROR/WARN/INFO/DEBUG)分类统计
awk 解析 syslog 格式      →  提取时间戳、主机名、服务名
find -mtime | tar -czf   →  日志归档压缩
trap SIGINT SIGTERM      →  优雅退出
```

### 3.5 模块五：主控调度中心

**技术核心：进程管理 + 任务调度**

```
dialog/whiptail  →  TUI 图形交互界面
crontab          →  定时自动巡检
nohup + &        →  守护进程后台模式
PID 文件          →  进程生命周期管理
HTML 模板拼接     →  巡检报告生成
```

## 四、技术要点总结

| 操作系统概念 | 实践对应 |
|-------------|---------|
| `/proc` 虚拟文件系统 | 读取系统状态信息的接口 |
| 进程管理 | `ps`/`top` 查看进程，`nohup`/`&` 后台运行，PID 文件管理 |
| 文件系统 | `df`/`du` 磁盘配额，`find` 文件检索，`stat` 权限查看 |
| 用户权限 | `/etc/sudoers` 配置审计，SUID/SGID 特殊权限位，World-Writable 检测 |
| I/O 重定向 | `>/>>/<` 日志输出重定向 |
| 管道通信 | `|` 管道连接命令，`tail -f | grep` 实时流式处理 |
| 信号机制 | `trap` 捕获 SIGINT/SIGTERM，守护进程的 kill 信号 |
| 任务调度 | `crontab` 定时任务，`sleep` 循环周期巡检 |
